Usa este artículo cuando ingreses por primera vez o cuando necesites confirmar que tu acceso base está listo.

## Checklist inicial

1. Verifica que tu nombre aparezca en el menú de usuario.
2. Confirma que tu segundo factor esté activo si el entorno lo exige.
3. Revisa los módulos visibles en la navegación principal según tu perfil.
4. Abre **Acceso** para validar roles y permisos efectivos antes de operar.

## Recomendación de arranque

| Tarea               | Dónde hacerlo             | Resultado esperado                  |
| ------------------- | ------------------------- | ----------------------------------- |
| Revisar tu perfil   | Configuración > Perfil    | Datos personales actualizados       |
| Confirmar seguridad | Configuración > Seguridad | Contraseña y 2FA vigentes           |
| Revisar permisos    | Configuración > Acceso    | Roles y permisos efectivos visibles |

## Si algo no cuadra

- Si no ves un módulo esperado, revisa tu artículo de acceso y luego contacta a un administrador.
- Si tu sesión exige una acción adicional, complétala antes de intentar operar sobre datos.
