El listado de usuarios concentra la búsqueda, exportación y acciones rápidas sobre cuentas existentes.

## Qué puedes hacer desde esta vista

- Buscar por nombre o correo.
- Filtrar por rol o estado.
- Exportar el conjunto visible o copiarlo para Excel.
- Abrir acciones por fila para editar, activar, desactivar o reiniciar acceso.

## Antes de actuar

1. Aplica filtros para reducir el alcance del cambio.
2. Verifica si la cuenta está activa o inactiva.
3. Revisa los roles asignados antes de cambiar permisos.

## Señales útiles

- **Activo**: la persona puede ingresar según su configuración de seguridad.
- **Inactivo**: la cuenta se conserva, pero no debe operar.
- **Sin roles**: revisa si el acceso efectivo será insuficiente para la tarea esperada.
